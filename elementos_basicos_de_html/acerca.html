Enviar email

El prefijo mailto: indica que el enlace abrirá el cliente de correo electrónico del usuario para enviar un correo.

informes@globalmentoring.com.mx: Esta es la dirección de correo electrónico a la que se enviará el mensaje.

?subject=Contacto: El signo de interrogación (?) inicia la cadena de consulta que especifica parámetros adicionales para el correo. Aquí, subject=Contacto establece el asunto del correo como "Contacto".

&body=prueba email: El ampersand (&) se usa para separar diferentes parámetros en la cadena de consulta. En este caso, body=prueba email establece el cuerpo del correo como "prueba email".

Enviar email: Este es el texto que se mostrará en el enlace. Los usuarios verán "Enviar email" y al hacer clic en él se abrirá su cliente de correo electrónico con los detalles ya completados.
